Help

How to navigate the home screen

On the home screen, you can select channels on the left hand side. Selecting a channel will load all the posts within that channel, displayed in chronological order. The events channel displays calendar events.

How to create channels and events

On the bottom of the home screen, there are icons for creating a channel ('+') and creating a calendar event (calendar icon). Selecting either of these will pull up a box to input information and create either a channel or an event.

How to navigate settings

To get to settings, select the gear icon in the lower left side of the home page. Once in settings, you can edit your profile, change display preferences, delete your account, or log out. Navigate to these options on the left hand navigation bar. To return to the home screen, click the back arrow icon on the bottom of the left bar.

Account and login help

From the login page, enter username and password to log in. Sessions are active for 15 minutes. If you need to create an account, navigate via the button in the bottom right to create account. Enter user info, create your account, then log in. You can change your email address, profile pic, and password in settings. You can also permanently delete your account and log out.
